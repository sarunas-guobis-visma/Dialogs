
## Dialogs

This repository covers 4 types of patterns to manage dialogs.

1. Basic state
2. Inversion of control 
3. Dialog manager 
4. Location

### How to start

Install dependencies
```
npm install
```

Start the project
```
npm start
```

Open the browser and go to
```
http://localhost:3000
```

### How to run tests

Run all tests
```
npm test
```
